About Backlog Tracker

Backlog Tracker was built to provide a simple way of keeping a backlog of your favourite shows across various streaming services

This project was built using the following technologies listed below:

  • HTML5
  • CSS3
  • Javascript
  • Bootstrap
  • SQLite
  • Django

Ultimately, the project was made possible thanks to the TVMaze API.

This is an open-source project so if you are interested in the source code, feel free to check it out here
